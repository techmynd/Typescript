install globally
================

> yarn add typescript
or
> npm install -g typescript

check if its installed
> tsc

// create tsc copnfig file
> tsc --init

// watch tsc file
> tsc --watch app.ts

navigate to folder that has scripts.ts and run compiler to have scripts.js copy
>tsc scripts.ts
